## Overview

This repository contains the source code for my personal engineering portfolio.  
It highlights projects involving:

- RF and telemetry systems
- Custom PCB design
- Mechanical CAD and 3D printing
- Mobile and web application development
- Applied AI and computer vision

---

## Selected Projects

**Rocket Telemetry Ground Station**  
Custom PCB integrating ESP32 and RFM69HCW RF module for bidirectional rocket communication.

**Jetson Power Data Logger**  
ESP32-based inline power monitoring system using INA260 with microSD logging and onboard UI.

**Autonomous Sumo Bot (Top 3 / 20+ Teams)**  
Designed and fabricated competitive robot using ESP32, L298N motor driver, ultrasonic sensing, and line detection.

**Autonomous Fire-Extinguishing Robot**  
Sensor-driven mobile robot with servo-actuated suppression system.

**Custom InfiniTime Firmware Development**  
Ongoing customization of open-source smartwatch firmware on ARM Cortex-M4 (nRF52832).
